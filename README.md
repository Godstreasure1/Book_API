## BOOK_API

### Background Context
This API is designed for the purpose of keeping documentaries and records

### Packages
- Express
- Mongoose
- Nodemon
- Dotenv


### Usage
> clone this project

> create .env file and enter the below:
```
PORT=****
MONGO_URI=***********
```

> install the necessary packages: `npm install`

> run the server : `npm start` or `npm run dev`
